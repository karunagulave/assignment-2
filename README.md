## 📌 Project Overview

This project demonstrates **Exploratory Data Analysis (EDA)** on an **Online Food Delivery Dataset**.

The objective is to analyze customer demographics, occupations, monthly income, feedback, and geographical distribution to uncover valuable business insights that can help an online food delivery company improve customer satisfaction and operational efficiency.
